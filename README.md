# Customer Segmentation Project

## Project Overview
Customer Segmentation is the process of dividing customers into groups based on their behavior, demographics, and purchasing patterns.

## Objectives
- Analyze customer purchasing behavior
- Identify customer groups using clustering techniques
- Generate targeted business insights
- Visualize customer segments

## Tools Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Power BI

## Features
- Customer Data Analysis
- Customer Segmentation using K-Means Clustering
- Purchase Pattern Analysis
- Customer Preference Analysis
- Segment Visualization

## Customer Segments
1. High Value Customers
2. Regular Customers
3. Occasional Customers
4. New Customers

## Expected Outcome
The project helps businesses understand customer behavior and improve marketing strategies through data-driven customer segmentation.
